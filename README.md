# 🛒 BlinkIT Grocery Sales - Power BI Dashboard

![Dashboard Overview](Blinkit_dashboard_overview.png)

---

## 📌 Project Overview

This project presents an interactive **Power BI Sales Dashboard** built using BlinkIT Grocery Data. It provides deep insights into sales performance, item analysis, outlet performance, and customer ratings — helping stakeholders make data-driven business decisions.

---

## 🎯 Objective

To analyze BlinkIT's grocery sales data and visualize key business metrics through an interactive Power BI dashboard, enabling quick identification of trends, top-performing outlets, and product insights.

---

## 📊 Key KPIs & Features

| KPI | Description |
|-----|-------------|
| 💰 **Total Sales** | Overall revenue generated across all outlets |
| 📈 **Average Sales** | Average sales per item/outlet |
| ⭐ **Item Ratings** | Customer ratings for grocery items |
| 🏪 **Outlet Performance** | Performance comparison across different outlet types and locations |

### 📉 Charts & Visualizations
- Total Sales by Outlet Type
- Average Sales by Item Category
- Item Rating Distribution
- Outlet Size vs Sales Performance
- Fat Content Analysis (Low Fat vs Regular)
- Sales Trend by Outlet Establishment Year

## 🚀 How to Use

1. Clone or download this repository
   ```bash
   git clone https://github.com/Purushothamreddy6749/PowerBi-dashboard__project.git
   ```
2. Open **Power BI Desktop** on your system
3. Open the file `Blinkit_Sales_Dashboard.pbix`
4. Explore the interactive dashboard — use slicers and filters to drill down into the data
